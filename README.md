# Hax_extend
Hax.co.id 自动续期
Forked from imlaoyou2333/Hax_extend

# 修改
使用Github Secret存储用户名和密码，无需创建私有仓库也不会暴露个人信息

# 用法
- 1.Star本项目
- 2.Fork本项目
- 3.在自己的项目中，前往Settings-Secrets-Action，新建两个名为'HAX_USER'和'HAX_PASS'的Secret，Secret值分别填入用户名和密码
- 4.启用Action
